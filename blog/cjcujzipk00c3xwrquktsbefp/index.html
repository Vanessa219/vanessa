NetBeans Weekly News # 215 - Oct 23, 2012

发表于 2012-10-23
















































2012-10-23

NetBeans IDE 7.2.1 发布
NetBeans IDE 7.2.1 是由 NetBeans IDE 7.2 更新而来的，它包含对 JavaME 3.2 SDK, JavaFX 2.2 SDK 的支持并集成了最新的补丁。


 
2012-10-23插件：phpcsmd
该插件通过 phpcs, phpmd, phpcpd 和 pdepend 任务及标注生成代码测量报告。		 
2012-10-23为第20界 NetBeans 管理委员会投票
Toni Epple, Glenn Holmer 和 Hermien Pellissier 作为下一届 NetBeans 管理委员会的候选人。花一点时间来阅读相关事项并投出你的一票！		 
2012-10-23NetBeans 博客：Dustin Marx
Dustin Marx 有一个非常好的博客，其中包含许多涉及到 NetBeans IDE 技巧、诀窍和见解的文章。通过阅读他博客中的技巧和诀窍来为自己写一个类似的原创 NetBeans 博客。		 
2012-10-23CERTivity 密钥库管理器1.1发布
EduLib 团队宣布基于 NetBeans 平台的新版 CERTivity 密钥库管理器发布。		 
2012-10-23媒体软件更新
在 Praxis 上阅读有关 NetBeans 平台上创建的开源媒体播放软件的最新状态报告。		 
2012-10-23Rob Terpilowski 在 JavaOne 大会的行程报告
Rob Terpilowski 是 JavaOne 大会上 NetBeans 的发言者之一，他将围绕 NetBeans 作为主题来进行经验分享。		 
2012-10-23从 JSF 到 HTML5
在某些情况下，考虑在 Java EE 应用中使用 HTML5 替代 JSF 前端是有意义的。这篇博客为您提供您所需要的所有资源。		 
2012-10-23加入 Play 框架支持 NetBeans
一个对 NetBeans 支持 Play 2.0 的新项启动了。阅读项目介绍，如果你想参与这个项目请在评论中添加你的名字。
