NetBeans Weekly News Issue # 276 - Mar 11, 2014

发表于 2014-03-11

NetBeans 8.0 准备好发布了吗？
NetBeans 8.0 GA 指日可待！但首先，需经过 NetBeans 社区的验收调查并让 NetBeans 团队知道 8.0 是一个非常棒的最终版本。调查截止日期为 3 月 11 日，周二。


加入 Java 8 的网上直播 - 3 月 25 日



Oracle 和 Java 开发者及合作伙伴社区加入了 Java 8 的启动。通过网上直播演讲和超过 35 个截屏，聆听，提问和学习来了解 Java8 如何帮助您共创未来。


免费的 Java Airhacks 事件 - 慕尼黑，3 月 26 日
Java EE 专家 Adam Bien 想听到你最困难，最具挑战性的 Java 问题。在慕尼黑机场和他一起参与一个免费的“airhacks”会议，围绕 Java EE，SE 和 FX 等 Java 相关的事情。


2014 JavaOne 大会开放征集提案
不要封锁你的思想 - 在 JavaOne 会议上分享他们！2014 年开放征集提案现已开放。


Java 桌面正处于手机为中心的世界？
最新出版的《NetBeans 平台入门》的作者 Jason Wexbridge 和 Walter Nyland，讨论了他们写书的原因及他们关于 Java 8 的看法。


书评：《NetBeans 平台入门》
“Jason Wexbridge 和 Walter Nyland 做了一个非常棒的工作，以基本原理和基础概念为桥接开始实际应用。”阅读 Vaughan Schmidt 的博客了解更多！


Emil Stoyanov：五个我最欢的 NetBeans IDE 功能！
Maven 依赖管理和本地文件历史只是 Emil Stoyanov 最喜爱的 NetBeans 功能中的两个。在文章中您将获取其余他喜欢的功能。


技术提示：NetBeans 头部许可
Gualtiero Testa 概述了如何在 IDE 中创建自定义授权头。


视频：NetBeans 代码补全的弹出列表框
Robert Liguori 演示了如何以及何时打开 IDE 的弹出列表，还包括基本的导航提示。
