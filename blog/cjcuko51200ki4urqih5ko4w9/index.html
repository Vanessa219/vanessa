NetBeans Weekly News Issue # 265 - Dec 24, 2013

发表于 2013-12-25


































NetBeans 对 iOS 版的 JavaFX 进行了支持！


NetBeans IDE 7.4 中 iOS 版本的 JavaFX 插件已经可以使用。它包含两个应用程序模板：一个传统的HelloWorld 和一个名为 GoldenFish 的模版，后者较为复杂且有漂亮的动画。

专访：NetBeans IDE 的 Vaadin 团队


“看起来真的从 NetBeans IDE 中获得了引导”来自 Vaadin 团队的 Sami Ekblad 写到。读他的访谈了解更多详情！

Josh Juneau：我最喜欢 NetBeans IDE 中的五个特性！


Java EE 专家 Josh Juneau 在他的文章中讲述了他喜欢 NetBeans IDE 的五大原因：对 Maven，版本控制工具，HTML5 开发的大力支持等。

Mark Stephens：我最喜欢 NetBeans IDE 中的五个特性！


Mercurial 和 NetBeans IDE 那干净的用户界面是我喜欢的特性中的一些。来自 IDRSolutions 的 Mark Stephens 在他的文章中分享到。


Aspose 团队宣布 NetBeans Aspose 插件








Aspose 团队宣布 NetBeans IDE 7.4 版本的 NetBeans Aspose 插件已通过 NetBeans 社区的验证。现在 NetBeans 插件管理器中已经可以使用。
