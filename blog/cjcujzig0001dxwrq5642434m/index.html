3. CSS Lint


http://csslint.net/


o(︶︿︶)o 唉，好好的看了下这个网站下面的选项，觉得自己悲剧了。平时咋就没注意这些东东呢，更可悲的是最近用 ！important 还用上隐了?



  •  Don’t use adjoining classes

  •  Beware of broken box model

  •  Use compatible vendor prefixes

  •  Use properties appropriate for display

  •  Disallow empty rules

  •  Don’t use too many floats

  •  Don’t use too many web fonts

  •  Don’t use too many font sizes

  •  Don’t use IDs

  •  Disallow !important

  •  Use vendor prefix properties correctly

  •  Don’t use units for 0 values
