NetBeans Weekly News # 179 - Jan 31, 2012

发表于 2012-01-31






























































 

NetBeans IDE 支持 PHP 5.4
下一个版本的 Netbeans 将会支持 PHP 5.4。可以在每日构建的 Netbeans IDE 中尝试编辑例如 Array Dereferencing 等新特性。


 


 
 

NetBeans IDE 中编辑器标签的多种摆放方式
这个迫切的特性在每日构建的 NetBeans IDE 中已经可以使用。可查看细节和截图。


 


 
 

NetBeans 7.1 中 Scala 插件可用
下载含有 Scala 插件的最新版本的 NetBeans 7.1。


 


 
 

虚拟开发者日：WebLogic（欧洲：2月7日，亚洲：2月9日）
Java 开发人员和架构师可以在动手实验室和课程中学习如何使用 Java EE 6, Oracle WebLogic Server 12c, Oracle Coherence Pack for Eclipse, NetBeans, Maven 和 Hudson.并可与 Oracle 产品专家进行在线聊天互动。


 


 
 

视频：NetBeans IDE 支持 PHP Smarty
了解如何使用 NetBeans IDE 支持 Smarty PHP 框架。


 


 
 

新的 Oracle Java SE 7 认证系列
今年需要一个新的目标？新的 Oracle Java SE 7 认证系列可提升你的技能，让你获知如何编写高效的 Java 代码。


 


 
 NetBeans IDE: 在 Swing 应用程序中使用JLayer
该教程演示如何在 Java Swing 应用程序中使用 JLayer 组件。		 
 

技术提示：在 NetBeans 7.1 中搁置和取消搁置的变化
John Yeary 演示了 NetBeans IDE 7.1 中一个很酷的新功能：允许开发人员在没有提交项目到源代码控制系统的情况下修改项目。


 


 
 

技术提示：使用 NetBeans IDE 导出快捷键设置为 HTML
Geertjan Wielenga 揭示了一个隐藏在 NetBeans IDE 中的功能：如何导出你自己的快捷键设置为 HTML。


 


 
 

NetBeans IDE 7.1 预览：NetBeans 平台支持
NetBeans 7.1 中一个新的增强的 NetBeans 平台功能—— Window 系统，Pluggable Multiviews 等。


 


 
 

Oracle 和 Java 生态系统
InfoQ.com 反映了 Oracle 下 Java 生态系统的状态—— JCP, OpenJDK 等。


 


 
 

B3log Solo 0.4.0 将于2012年2月19日发布
GAE 博客 —— B3LOG Solo 0.4.0 将于2012年2月19日发布！这是 0.4.0 的正式版，主要是加入了移动皮肤支持，并修复了一些 Bugs。
