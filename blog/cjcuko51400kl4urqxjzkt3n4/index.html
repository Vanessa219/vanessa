NetBeans IDE 可支持 PHP 5.5


NetBeans 8.0 测试版支持 PHP 5.5，包括 Generators, Class Name Resolutions 以及其他。NetBeans PHP 博客作为团队将在未来几周内揭示关于新功能的详细信息。

NetBeans IDE Codename One 中五个很酷的特性


Codename One 允许 Java 开发人员构建 iOS，安卓，Windows Phone，RIM和其他设备上的原生应用。本文介绍了最近发布的 2.0 版本给 NetBeans 用户带来的新特性。
