视频：NetBeans UML插件预览

由 Zoran Sevarac 的领导，在 Belgrade 大学的开源团队正在开发一个 NetBeans UML 插件。该截图展现了该插件可用特性的一个预览。

插件：NBAndroid

NBAndroid 提供了对 Android 应用周期开发的支持。它在 NetBeans IDE 中的功能涵盖了 Android 开发的各个方面。

在 Java 平台，企业版7中使用 WebSocket 进行实时通信

本教程和视频向您展现了如何使用 WebSocket API 在客服端和服务器端创建实时通信的应用程序。

新的发布：jMonkeyEngine 3.0 初学者指南

对游戏开发有着野心的开发者们可以在本书中学习到如何使用 Java 构建 3D 游戏模型，该书包含了最佳实践、技巧和窍门，以及代码示例。

NetBeans 7.3.1 对 Java EE 7 的支持

NetBeans IDE 为 Java EE 7 提供了全面的支持，包括工具、模版和构建 Java EE 7 应用程序的示例。Arun Gupta 对最新的 NetBeans 更新版本中的新功能进行了概述。

发布：NetBeans 平台上开源的航海导航软件
