 提交主题或新奇的想法到 NetBeans Poll！
开发人员可以有自己的团队！NetBeans Poll 可帮助 NetBeans 用户和开发者快速了解他们认为的当前技术趋势及 NetBeans 社区主题。我们非常欢迎这样的建议。严肃或轻松愉快、主题或新奇的，请把您的想法转达给我们。		 
 插件：NetBeans IDE 中的 Drupal
NetBeans Drupal 开发工具是 NetBeans IDE 中用来帮助 Drupal 模块开发和主题开发的一个模块。它提供了快速访问所有核心 Drupal hooks 和主题的覆盖方法。		 
 新的 NetBeans 工具–HTML5

 GlassFish 中的 Parleys 和 Devoxx

 在 NetBeans IDE 中使用TomEE

 使用 NetBeans 搭建的新型液压建模软件
Fudaa-Crue 是由 CNR（Compagnie National du Rhône）专门指定的液压建模软件，CNR 是一家致力于能源生产，导航，并提供灌溉及其他农业用途的公司。		 
 视频：EJB 依旧对 Java EE 7 感兴趣？
观看 Adam Bien 关于 EJB 3.2 和 CDI 托管 bean 结合的最新截图。了解他们如何在 Java EE 7 中仍然提供最精简的可选项！		 
 重写 Feed 阅读器教程

 Novi Sad 的 NetBeans 用户社区

 在 Facebook 上加入 NetBeans 组
