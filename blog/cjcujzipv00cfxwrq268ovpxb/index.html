 在 EJB 3.1 中创建异步方法
在 本教程中，您将创建一个 Java EE 6 的 Web 应用程序并添加下列组件到里面——一个无状态会话 Bean 与两个异步方法。我们定义了一个 Servlet 来调用异步方法并进行跟踪调用和完成的次数来演示异步方法调用的特性。 index.jsp 将包含一个表单提交按钮，让你来执行应用程序的运行。表单将提交给 Servlet 去调用定义在会话 bean 中的异步方法并返回从定向的页面 response.jsp。展现异步处理程序的相关信息给用户。
