Google chrome 浏览器试用后简单的比较

发表于 2008-09-04


转自http://tryonmind.javaeye.com/blog/235954


Google   的浏览器面世了，下载下来用了一下，界面简洁明快，地址栏又可以当搜索栏用倒是省了不少事。直接点击小五星添加标签也方便，新打开的标签直接是常访问的页面导航。


速度还算可以，有时感觉还没有ie快。

功能设置很少，还得注意好保护隐私，要不可能都给google拿走了。

内存使用上，刚启动后firfox>google chrome>ie8（可能是我firfox插件装太多了吧）,在同时打开csdn，javaeye，163，sohu后正好反过来。firefox<google chrome<IE8。

不过对于习惯了直接在地址栏书地址打开新的页面，习惯了拖拽，习惯了在无关页面后直接点“最近浏览”找回的我来说用起来挺不习惯。

chrome倒是也有js的debug，不过好像很简单，和ie8和fiefox的插件的功能比起来实在差太远。不过有google的支持，又开源，以后会很有发展。


下载地址：  http://www.google.com/chrome/


 


IE8   bate1原来装上后问题太多，进入163邮箱一点就死，昨天正好把ie8升 级到 bate2，测了一下，还不错没发现什么问题，建议大家去做升级，不过据说装上后原来基于ie核心的一些浏览器不能正常启动了，不过我一直用的腾讯TT没 事。IE8的 js debug功能也相当强大，可以和firefox的firdebug媲美。兼容性试图，保护隐私方式浏览，加速器都挺好，同样有快速导航视图。还是期待正 式版出来，IE还是牛，微软的东西就是好用。 


 


Firfox   原来也一直没怎么用，只是在调试程序的时候用一下，这两天算是开始喜欢上它了，这主要源于firfox有那么多强大的插件的支持，经过插件的武装后用起来还是蛮舒服的，也特别方便。做页面开发一定少不了firfox，有了这么多插件，平时上网也少不了啦。


 


g oogle chrume能从IE、Firfox那分一杯羹吗？拭目以待吧。


 


 


最后给大家推荐几个好用的插件：


ColorfulTabs： 让firefox的tab加上颜色，视觉效果不错！


DownthemAll：让你很轻松的下载浏览的网页资源


firebug：这个不用说了，必备的


IE Tab：把firfox临时变成ie，用ie来显示页面，避免一些网站显示错乱或显示不出来


JS View：很好用，直接可以用它查看到浏览的网页引用到的js文件和css文件


Live  HTTP headers：查看http头的信息


RemindFox：备忘录


Web developer：页面设计的好帮手


All-in-one Sidebar：提供一个方便的侧边栏


Piclens：一个相当牛，相当炫的东东，可以直接搜索一些网站的图片新闻，视频等用3D的炫效果展示出来。


 


Piclens的绚烂效果，像一面墙，可以用鼠标拖动，点击放大
