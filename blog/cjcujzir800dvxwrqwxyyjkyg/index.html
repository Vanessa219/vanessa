NetBeans 时事通讯（刊号 # 142 - Apr 14, 2011）

发表于 2011-04-14

项目新闻


NetBeans IDE 7.0 候选发布版2可用


NetBeans 团队很高兴地宣布 NetBeans IDE 7.0 的第二个候选版发布。下载 RC2 版本，可通过 NetBeans 邮件列表和论坛，Twitter，或提交报告给我们进行反馈。NetBeans 7.0 最终版计划于四月发布。





新版 Agavi 插件发布


Markus Lervik 宣布 NetBeans Agavi (PHP5 应用程序框架)更新。




LIVE Oracle TechCast：NetBeans IDE 7.0（4月12日，星期二）


NetBeans IDE 7.0 快发布了！但在发布之前，我们正在征求 NetBeans 用户的一些建议。参加 NetBeans Engineering Manager John Jullion-Ceccarelli 主持的关于 NetBeans IDE 7.0 在线研讨会 LIVE Oracle TechCast，主要讨论对 Java SE 7 的支持及 IDE 的其它改进。相关问题请关注 Twitter: hashtag #techcastlive。准备好你的问题并加入在线讨论吧！日期：4月12号 星期二，10：00 pst(Check your time zone)
