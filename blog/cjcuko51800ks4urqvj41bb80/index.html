NetBeans 播客 69 - 嵌入式版本
Oracle 技术传道者 Terrence Barr 和 Simon Ritter 讨论了 Java8 的嵌入式平台，SE 和 ME 之间嵌入式的差异，以及由 Oracle 提供的免费培训。


 


 

Java ME 8 的示例和演示代码
Terrence Barr 为开始学习 Java ME 8 的开发者收集了一些好的资源。


 


 
 

视频：NetBeans 中的拖拽
该示例展现了如何使用 NetBeans 中的拖拽功能轻松创建用户图形界面。


 


 
 

使用 NetBeans IDE 开发 Grails
在 NetBeans IDE 中只需 5个简单的步骤，就可快速学习 Grails。


 


 
 

Hippo CMS 和 NetBeans IDE
Hippo CMS 是一个基于 Java 的开源内容管理系统，它可以让开发人员动态创建及定制个性化的内容。了解如何使用 NetBeans IDE 创建一个 Hippo CMS。


 


 
 

为什么 NetBeans IDE 非常适合 Java 教学（第6部分）
保加利亚索非亚大学的 Milen Petrov，评论了 NetBeans IDE 中的一些功能，例如 GUI 原型设计，重构，原码生成器和其他方面可以让学习编程更加简单，能够让学生有更多的创造性和兴趣。
