NetBeans Weekly News Issue # 280 - Apr 15, 2014

发表于 2014-04-15

插件：NetBeans 的 Scala 插件
Caoyuan Deng 已经针对 NetBeans8 升级了 Scala 插件！今天就开始使用吧。


NetBeans IDE 8.0 中的 Python
在 NetBeans 8 中使用 NetBeans-Python 插件构建 Python 项目。


资源库与NetBeans8 和 PrimeFaces
Josh Juneau 会教如何在你的应用中使用 PrimeFaces 引入一个布局组件并简单选择最适配的布局。来看看在 NetBeans 8 中如何使用它吧。


调试 Maven Mojo 插件
Benno Markiewicz 详解了如何利用 NetBeans 8 正确高效地配置 Maven Mojo 插件。


NetBeans 8 创建 Nashorn 应用
在 NetBeans 中使用 Java SE 8 中的 Nashorn 特性创建一个应用。


2014 年 3-4 月的《Java Magazine》
最新一期的《Java Magazine》已经出版了。从 Java 领域专家的文章中可详细了解 Java 8 的最新特性，而且《Java Magazine》是免费的！


新的 NetBeans 博客
很受欢迎的博客作者 David Salter 在他的博客中新建了一个关于 NetBeans 的类别。阅读可了解他在 WildFly, JBoss, TomEE 等新插件的工作。


NetBeans 社区推文：@rchouinard
刚升级到 NetBeans 8.0。感觉像 @netbeans 团队的一个新版本。



NetBeans 社区推文：@devronbaldwin
@netbeans 非常棒，在很多时候都会让你感到“太棒了，他既然有这个功能！”。@GeertjanW 和他的团队都非常惊奇。
