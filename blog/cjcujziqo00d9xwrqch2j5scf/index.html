NetBeans Weekly News Issue # 331 - Jun, 12, 2015

发表于 2015-06-12

新功能：JavaScript 高级故障排除工具
工作中使用 JavaScript 吗？NetBeans 的下个版本加入了一个新的特性，Knockout.js 分析工具应用。在这里可以快速获取它的概述并了解 到 NetBeans IDE 8.1 中计划的强大新功能。


新插件：HTML5趣味包
你知道自 HTML5 发布后，在 HTML 中有一个“视频”和“音频”的标签吗？你知道现在原生的 HTML 对日历，代码完成和拖放的支持吗？安装 HTML5 趣味包到 NetBeans IDE 中，然后就可立即体验 HTML5 应用程序！


6月16日：Java EE 和 GlassFish 伦敦用户群
加入伦敦 Java EE 和 GlassFish 用户群玩转 Payara Micro。在英国伦 Skills Matter 免费托管，这个动手工作坊会给你一个独特的机会去玩转最新版本的 Badass Fish – Payara Micro。


6月22日：巴西 NetBeans Day
免费领取巴西 NetBeans Day 的门票，届时将有 Bruno Souza，Geertjan Wielenga，Bruno Borges，以及更多的演讲者加入到从 Java EE 到 JavaScript 以及两者间任何范围为主题的讨论。


本周 Tweet：@gegastaldi
让人高兴的是，@NetBeans 插件门户上现在有 @JBossForge 类似的插件了：http://plugins.netbeans.org。赶快使用它！


Facebook 上的 NetBeans
在 Facebook 的 NetBeans 页面上可以关注最新的新闻，每天都有新的内容发布，包括提示和技巧。


Zeef.com 上的 NetBeans
在巴黎， Nebrass Lamouchi 向您介绍了 Zeef.com，在那里你会发现和创造您最喜爱的有关 NetBeans 主题链接的列表。该页面包含了很多有用的 NetBeans 新闻，文章和教程。同样也有很多的博客链接和有用的 NetBeans 书籍。


YouTube：NetBeans 平台上的银河系漫游指南
注意到 NetBeans 在平台上推出了最近的 JavaOne 会议——你可以看到 NetBeans 平台正在推出的几款应用程序！在过去几天这些发布到 YouTube 上的视频都是免费的。
