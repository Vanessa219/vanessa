NetBeans Weekly News: #125-Nov 17,2010

发表于 2010-11-17
















































2010-11-17NetBeans IDE 6.9 的第二个补丁现在已可使用
NetBeans 团队已经发布了一个 NetBeans IDE 6.9.1 更新的补丁。该补丁包括了 C/C++ 模块，IDE 平台，Java，Java Debugger，Java Web 应用程序，Maven，移动，OSGI，PHP 和 RCP 平台上的一些 bug 修复。
要获得修复，必须安装和运行 NetBeans IDE。同样，您也可以通过下载 IDE 的插件管理器来进行修复。		Edit
2010-11-17NetBeans IDE 7.0 支持 JDK 7
可在 NetBeans IDE 7.0 中获取 JDK 7 的概述和图像支持。		Edit
2010-11-17插件：Javeleon
Javeleon 允许开发人员动态透明地更新正在运行的开发中应用程序的 test-instance 中的 NetBeans 模块的 Java 代码。		Edit
2010-11-17在 NetBeans 电子商务教程——测试和分析
本教程是一套 NetBeans 电子商务系列的演示，教您如何使用 JMeter 和 NetBeans Profiler 来检查应用程序的功能和性能。		Edit
2010-11-17NetBeans IDE 在悉尼（11月17日）及墨尔本（11月19日）的 OTN 开发者日
Oracle 技术网（OTN）开发者日是一个由企业 Java 开发者专门为自由开发者设计的活动，让所有的参加者学习如何丰富他们在最新发展的 Java 技术中的开发经验。参与在悉尼和墨尔本的 NetBeans 传道者 Ashwin Rao 将演示 NetBeans IDE 对 JDK 7 和 Java SE 7 的支持。		Edit
2010-11-17在 Ubuntu 上使用 NetBeans 开发 Android 的配置
NetBeans 用户 Tiklu 书写了他的经验：在 Ubuntu 操作系统上使用 NetBeans IDE （7.0 M2）建立自己的 Android SDK。		Edit
2010-11-17技术提示：NetBeans IDE 的 CVS 压缩
来自 NetBeans 梦之队 Michel Graciano 保存的邮件列表中一个过时但有用的提示。		Edit
2010-11-17使用 NetBeans IDE 和 Maven 进行 Java 开发
该系列博客由三部分组成，介绍了如何使用 NetBeans IDE 配置 Maven。		Edit
2010-11-17基于 NetBeans 平台开发移动互联网门户
Amdocs ChangingWorlds 是一家电信软件公司，致力于为世界各地的大型电信公司开发移动互联网门户网站软件。使用NetBeans 平台，Amdocs 公司开发出一种模板生成器，它允许客户定制和维护他们的门户。		Edit
