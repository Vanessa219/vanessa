在 Oracle’s Java7 Summit 上贡献您的 Java 专业知识


Oracle’s Java7 Summit（11月2日-11月4日）将提供 Java 开发人员和架构师一个深入的关于最近发布的 Java 7 和 OpenJDK 技术培训。与2011欧洲 EclipseCon 一同，Java Summit 将有来自 Oracle 和 OpenJDK 社区的专家进行主讲。Summit 将邀请 Java 专家进行研讨。截止日期为8月17日。




征集提名：Oracle Java 2011 创新奖
