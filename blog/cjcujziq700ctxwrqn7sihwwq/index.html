NetBeans Weekly News # 500 - Aug 16, 2011

发表于 2011-08-23

NetBeans 7 满意度调查仍在继续！
谢谢所有那些完成 NetBeans 7 满意度调查的人们！如果您还没有参与——我们非常强烈地鼓励您进行参与。只需要大约5分钟的时间，本次调查为您提供了一个真正的机会：“指引” NetBeans 项目的方向。

梦之队：“第一个补丁”程序
来自 NetBeans 梦之队的 Tom Wheeler，现已正式公布了第一个补丁程序，旨在帮助你开始为 NetBeans IDE 贡献补丁。

AgroSense Farm Manager 1.0.9
AgroSense Farm Manager 新版本是基于 NetBeans 平台的 Maven 项目。这一次，包含了一个 基于 Flamingo 的工具栏。你想为项目做出贡献吗？它是开源的并且欢迎您的加入。

NetBeans API：org.netbeans.editor.SideBarFactory
一系列的介绍让您简单的了解 NetBeans APIs 后，来自 Denmark 的 NetBeans 平台的开发人员 Andrzej Olszak 为您展示 SideBarFactory 及其代码。

PHP 博客：为 Internal S/FTP 客户端保持连接
NetBeans PHP 团队在他们非常活跃的博客上公布了 PHP 的又一新特新。对于您的远程服务器，上传并保存的功能现在更快更友好了！

NetBeans IDE 7.0 中的 JUnit
NetBeans IDE 7.0 中的 JUnit 有什么新支持？一个意大利的博客告诉你与之相关的内容。

Jaroslav Tulach：NetBeans IDE 7.0.1 的故事
NetBeans API 架构师讨论了一些关于 NetBeans IDE 7.0.1 中 bug 修正的有趣问题。

NbAndroid
来自 Indonesia 的 Tonny Kohar 教你如何在 NetBeans IDE 中开发运行 Android 程序！


GAE 博客 —— B3log Solo 0.3.0 正式版发布


GAE 博客 —— B3LOG Solo 0.3.0 正式版发布了！该版本重写了后台管理，对前台模版进行了调整优化，修复了一些 Bugs，并进行了一些改进以及新特性。

另外，0.3.0 加入了插件功能，可以通过插件来扩展功能了 ;-)
