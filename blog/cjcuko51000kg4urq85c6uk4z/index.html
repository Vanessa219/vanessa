NetBeans IDE 7.4 目前对 PHP 强大的单元测试框架——atoum 进行了支持。

在 JavaOne 2013 大会上，所有围绕 NetBeans IDE 和 NetBeans 平台的会议都被一一罗列出来了。据此，您可以对您的 NetBeans JavaOne 大会日程进行安排。

参加 JavaOne 大会的最佳提示

视频：Toni Epple 的专访

Java EE7：添加一个 JavaServer Faces 2.2 viewAction 的组件到 Web 中

整合 NetBeans Visual Library 和 JUNG


Visual Library 是在 NetBeans 中的一个库，他可用来创建一个优美的、复杂的，且具有移动布局的独立的 Java 应用。JUNG是一个可用来连接数据并进行图形化展现的库。Tim Boudreau 为您展现了两者组合的结果。

技术提示：Maven，Java EE 7，Backbone.js 的混搭

NetBeans 的 COMTOR 插件


COMTOR 提供了一个在 Java 源码注释中评估其质量和一些特性的基础设施。

网上直播：JVM 和 Java 垃圾回收器（2013年9月11日）
