Windows 8.1 预览版的折腾

发表于 2013-07-04


从去年病毒入侵后就没有这么折腾过了。享受折腾的感觉，有了折腾就有了吐槽，有了吐槽才能写博客






源起：



中午吃饭的时候，开着 IDE 撸了一把，中途电脑就熄火了，各种电风扇、电源、电池组合交差着试，他就像那么一坨铁一样死死的躺在那。







续之：



临时领了台电脑，加上最近手头工作也比较轻松，再加上最近在弄 B3log 的 Metro 皮肤，再再加上我是前端、我用 Lumia 920、我要跟上时代的潮流，HOHO，就让我来体验一下 Windows 8.1 吧。







安装：



下载安装包：WindowsBlue-ClientwithApps-64bit-ChineseSimplified-X1899611.iso



下载硬盘安装工具：nt6 hdd installer v3.0.8




papa click nt6 hdd installer v3.0.8 -> restart -> choose Mode 2 -> follow your heart








爱上了你：



安装完毕，开始体验。



开始界面和 Lumia 920 一样，可以随意拖动，马上就把他当 Win7 的桌面来用。令有怪癖的我最 happy 的是，我可以在桌面上只放一个回收站，哇哈哈哈。



再有就是左上角的提醒、win + F 的飞速搜索功能、应用商店、IE11、登录可使用图片密码、与微软帐号同步、支持 Google 的同步（读大学的时候不喜欢微软，不过现在慢慢的有点喜欢了，因为他在变）。。。还有 gmail 、日历什么的，统统都是手机、电脑同步，非常的同步。他里面的人脉、邮件、日历被分为高效工作，名字起的真好，真的是很高效，很方便、很喜欢。







意外的惊喜：



第二天中午去前台拿包裹，可是电脑还没送去修，想着突然就死了，会不会慢慢就复活了。死马当活马医吧，试一试。擦，真的就这么好了。哇咔咔，可是可是，这不是 Windows 8，我~我我~~~，再次重装吧。开动, go go go ->>>>>>



电脑只有一个盘，当然是要先分区咯。找了各种工具，不太理想，最后发现 Windows 7 是自带分区功能的，随用之。分好区，把重要的东西分别归类，很久没整理电脑了，不错不错，顺便整理下。就像1周不大扫除，心理不舒服、心情不清晰一样。重复安装步骤，bingo 







风平浪静中的折腾：



重装后，折腾了一个多小时，把需要的驱动、软件都装好了，微软帐号也同步创建好了。



突然看到以前装 Lenovo 的一个盘空了10 多 G，想和到另外一个盘上，继续用自带的磁盘管理工具。纠结乱弄了一会，发现搞不定，放弃之，那小盘就用来放无法分类的东西吧，就像老友记中 Monica 不准别人打开的那件屋子一样。



接着就把应用商店不用的应用卸载了，开始体验邮件和日历应用。邮件没什么问题，很好，可惜日历同步不了 google 的，MMY，好不容易装上了个 google calendar，人脉、日历和邮件却坏了。折腾许久后，删除了 Google Calendar（讨厌他的广告），人脉、日历和邮件在应用中出现了，可惜却无法卸载，点击了也没有反映。想着重启下，好了。



OMG，找不到系统文件，突然闪过了纠结磁盘管理工具时的情景，欲哭无泪。







夜晚 1点 半：



拷贝了 iso，和 Windows 8 USB Installer Makerr，准备回家重装。Windows 8 USB Installer Maker 查不多弄了半个小时，满怀激动之心插上，呜呜呜，什么世道，黑屏提示”不可识别的 XX Table”。



我折腾不动，看来只能先用原来做好的 Win7 USB 把系统装好，然后再装个 Windows 8.1。插上另外一个 Win7 的 USB 时，就自动引导进去。应该是引导的问题， win+R -> msconfig -> enter -> 点击引导，果然，引导里面啥也木有。速度使用 NTBOOTautofix，终于好了。



接下来又开始折腾人脉、日历和邮件了。。。既然打不开，从应用页面又卸载不了，只能 win+F 搜索 Windows Powershell，然后以管理员运行,



cd C:\Program Files\WindowsApps



ls



天啊，我咋知道人脉、日历和邮件是那一坨呢，看呀看，就你了



Remove-AppxPackage microsoft.windowscommunicationsapps_17.3.9431.0_x64__8wekyb3d8bbwe



然后重新在应用商店里面安装就 ok 了，至于同步日历的问题，我把 google 导到微软帐号上面。这样用一个也不错。只能怪你没什么特殊的吸引力了， So Sorry！



最后就是无限网络的问题，连不上。不知道是重装了无限网卡，还是把 wifi - 属性 - 配置 - 高级中的 802.11d 和 空闲时省电禁止了，反正最后是可以用了。














放下，投入更要你投入的地方：



折腾告诉我，即使你想到了，也未必是必须的。有些软件装了，折腾了，不一定就会用的到。有些你认为用不到的软件，删了，还得重装回来。



不要过于追求一定要各种兼顾，只要能达到目的，让自己使用的高效、Happy，放弃一些，有何不可？纠结于上翻滚了一圈，只是在浪费时间。








开工开工：



亲爱的系统，我要去写代码了，不纠结你了，即使你现在还有些不完善，就等我拖延到需要用你那不完善的地方再说吧，也需都用不到，我不就省了么，哇咔咔。







NOTE：



口水话吐槽了一篇，重点工具已被标红。其余你可以略过。
