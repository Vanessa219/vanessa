Infoworld发布2009年度开源软件大奖

Infoworld发布了2009年度开源软件大奖
，分别评选出：最佳开源开发者工具
| 最佳开源企业软件
| 最佳开源网络软件
| 最佳开源平台和中间件
。还包括，有史以来最伟大的开源软件
和十大开源软件名人堂
。名单如下：


最佳开源开发者工具：


BrowserShots
是一个非常优秀的开发者在线资源库


Drools

（JBoss Rules ）具有一个易于访问企业策略、易于调整以及易于管理的开源业务规则引擎，符合业内标准，速度快、效率高，是JBoss Rules的前身，由RedHat赞助。


Eclipse Web Services Tools
插件。


Hadoop and Hive
专门用于大数据处理。


jQuery
Javascript框架，太出名了，不介绍了。


Mono
一个开源.Net的实现，技术本身没有问题，但因在自由软件平台中实现商业性的.Net，使其变得很有争议。


NetBeans
Sun公司推出的开源IDE，还用介绍吗？


OpenStreetMap
,
是英国的Steven Coast发起的，他认为，ZF制图部门拿着纳税人的钱去制图，然后这些数据又收费，十分的不合理
（美国许多数据是免费的）。于是就搞了这一项草根地图运动：由个人行走、骑车、开车等方式记录GPS踪迹（trace），并上传整理制图，众人共享。目前
看来相当成功。



WebKit
是Web浏览引擎，被广泛用于iPhone, Android, Palm Pre以及Nokia等其它手持设备。



最佳开源企业软件：


Compiere
是一个通用的ERP包，是全球排名第一的开放源代码 ERP（集成 CRM）企业应用解决方案，被设计为适用于全球范围的市场。


Dimdim
是一个视频会议的软件系统。


Drupal
是一个非常出名的内容管理系统，不用再介绍了吧。



Intalio 企业级业务流程管理系统


JasperSoft BI Suite
, 一个非常有名的开源BI套件。


Magento
是非常著名的电子商务程序。


Openbravo
是一套适合于中小企业并且基于web可扩展的ERP系统。这个ERP系统所包括的功能可实现生产管理、仓库管理、销售管理、财务管理。同时内置CRM(客户关系管理)和BI(商业智能)。


Pentaho BI Suite
提供了一整套的商业智能（BI）能力，包括报表、分析、仪表盘、数据挖掘、数据整合等等，已经成为世界上最流行的开源BI
套件。


Piwik
是一个开放源代码的Web统计软件，基于PHP+MYSQL开发，可以完全替代Google确Analytics，它给你一些关于你的网站的实用统计报
告，比如网页浏览人数，访问最多的页面，搜索引擎关键词等等。Piwik拥有众多不同功能的插件，你可以添加新的功能或是移除你不需要的功能，Piwik
同样可以安装在你的服务器上面，数据就保存在你自己的服务器上 面。你可以非常容易的插入统计图表到你的博客或是网站或是后台的控制面板中。



Sugar CRM
最伟大的企业级开源CRM系统。功能强大，不介绍了。


WordPress
目前应该是最流行的BLOG平台，本站系统也是基于WP建立。


开源网络软件列表：



Cacti
Cacti是一套基于PHP，MySQL，SNMP及RRDTool开发的网络流量监测图形分析工具。它通过snmpget来获取数据，使用RRDtool绘画图形，而且你完全可以不需要了解RRDtool复杂的参数。


IPCOP
目前是地球上最强悍的开源防火墙系统之一，最早是基于Smoothwall开发，目前基于LFS发行，核心基于Netfilter/IPTables。拥有海量的插件，实在是太强大了，完全超越有些商用的系统。


Kamailio
是一个开源的SIP代理服务器。


KeePass
是一款轻量级的密码管理系统，强大易用的开源密码管理系统，使用该系统，你可以方便的对各种文件加密，你的密码或者key文件存在数据库中。


Nagios
是一个在linux及unix环境下运行的监控系统和网络的应用程序，它监控您所指定的主机和服务，并在情况变得更糟或更好时提醒您注意。


Openfiler
是一个非常流行的SAN/NAS 存储应用管理系统，基于rPath Linux，Openfiler支持IDE、SAS、SATA、SCSI、iSCSI接口。


OpenNMS
是一个功能非常强大的开源企业监控系统。


Puppet
是一种开源的IT自动化工具，他的发布一发不可收拾，包括Google、Twitter、Sun、Sony、Red Hat、NewYork Stock Exchange（纽约证券交易所）、Digg、SlideShare、Shopzilla、哈佛大学和斯坦福大学都在使用Puppet管理自己的系统。


Untangle
是一个非常出色的网络安全网关，包括反垃圾、防病毒、Snort以及7层过滤 ，几乎可以做到: 反垃圾、反病毒、反间谍软件、反网络钓鱼、广告拦截、协议过滤, 防火墙、IPS、 VPN、QoS。


最佳开源平台和中间件列表：


Jitterbit
是一个开源集成工具，它提供了一种快速、简单的方式来设计、配置、测试和部署集成解决方案。支持多种文档类型和协议包括：XML、web services、database、 LDAP、text、FTP、HTTP(S)等。


Mule
是一个企业服务总线(ESB)消息框架。


Nginx
是一个高性能的HTTP和反向代理 服务器，也是一个
IMAP/POP3/SMTP 代理服务器。它是由 Igor Sysoev 为俄罗斯访问量第二的Rambler.ru
站点开发的，它已经在该站点运行超过两年半了。Igor 将源代码以类BSD许可证的形式发布。尽管还是测试版，但是，Nginx
已经因为它的稳定性、丰富的功能集、示例配置文件和低系统资源的消耗而闻名了。


OpenVZ
是基于Linux平台的操作系统级服务器虚拟化解决方案。OpenVZ采用SWsoft的Virutozzo虚拟化服务器软件产品的内核，Virutozzo是SWsoft公司提供的商业虚拟化解决方案。


Talend Open Studio
是一个非常出色的数据集成项目。


Turnkey Linux
是一个基于Ubunt的发行版，目前包括了16个应用，分别是：AppEngine、Tomcat、WordPress、phpBB、LAMP、
Rails、Joomla、MediaWiki、Drupal 6、Drupal
5、LAPP、Django、MySQL、PostgreSQL、Core、Bootstrap。


VirtualBox
相当优秀的虚拟化软件，没有理由不使用它。


WSO2 Carbon
是 WSO2基金会的SOA 平台产品。Carbon框架的核心功能包括：数据服务，ESB/routing, 流程管理，注册，监控以及安全。


Xen
又一个虚拟化软件，这年代虚拟化应用太流行了。


十大开源软件名人堂列表：


Linux kernel
内核，不用介绍了吧。


GNU工具和编译器
，GNU工程有太多的应用软件，因为有了GNU才有今天的GNU/LINUX，正因为RSM和LINUS的全并，才有了今天繁荣的GNU/LINUX世界。


Ubuntu
，最流行的GNU/LINUX发行版，对桌面GNU/LINUX的普及有相当大的意义。


三个BSD（FreeBSD，NetBSD，OpenBSD），因为这三个BSD的存在，所以LINUX并不是唯一最好的开源操作系统。


Samba
是在Linux和UNIX系统上实现SMB协议的一个免费软件，由服务器及客户端程序构成。实现了与WINDOWS之间的文件共享。


MySQL
是最流行的数据库


BIND
是一个LINUX和UNIX平台下的DNS服务器软件


Sendmail
非常出色的邮件系统


OpenSSH和OpenSSL，为LINUX和UNIX的网络通信提供了安全加密，OPENSSH提供了远程连接服务，OPENSSL提供SSL加密通信。


Apache
是一个非常流行的WEB服务器。


(还差PHP/PYTHON/PERL，否则LAMP就在名人堂中团聚了)
