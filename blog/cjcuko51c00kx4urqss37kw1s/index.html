NetBeans Weekly News Issue # 296 - Aug 13, 2014

发表于 2014-08-13

用户眼中 NetBeans 的十大功能
从博客、文章和社交媒体中挑选出来自世界各地的 NetBeans 用户所分享自己在 IDE 中喜欢的功能，以及对日常开发工作和并学习新技术、语言能力的影响。


Tim Boudreau：我最喜欢 NetBeans IDE 的五个功能
了解什么是“Hippie Completion”可以帮你像 Tim Boudreau 一样，作为技术顾问，传播者和作者，描述自己最喜欢的 NetBeans IDE 的功能。


Leon LaSpina：为什么 NetBeans IDE 非常适合教学 Java
Leon LaSpina 切换 NetBeans IDE 来教授纽约市高中学生学习计算机科学。他分享了他的观点：是什么让 IDE 成为 Java 教学的理想工具。


Virtual JUG：自以为是的 JavaFX8 - 8月27日
加入有关 JavaFX 8 虚拟会议的开发者们。 Java 顾问专家 Adam Bien 将为您从应用角度来介绍 JavaFX 8，主要讲解结构，架构和（软件）设计。


视频：NetBeans 基础
一系列视频向初学者们介绍了 NetBeans IDE 及 Java 编程。


通过 NetBeans 在 Raspberry Pi 上部署 Java 应用程序
学会在 NetBeans IDE 上为 Raspberry pi 建立一个远程的 Java SE 8 平台上。附带的视频也可以。


NetBeans IDE 中无需编码的 Web 项目
了解如何在 NetBeans IDE 中创建一个无需编写任何代码的 Web 项目。


NetBeans 社区推文：@AtulPalandurkar
使用 @netbeans 进行 ＃HTML5 应用的开发和培训，这非常有趣，同时也非常容易执行许多任务。


NetBeans 社区推文：@abhideepchakrav
@netbeans 它是个很棒的 IDE。整个团队使用 Eclipse 工作，但我使用 NetBeans :) 。
