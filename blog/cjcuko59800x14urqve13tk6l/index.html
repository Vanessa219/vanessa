ubuntu上netbeans6.1在desktop application的小问题

发表于 2008-05-09

发现netbeans真的好强大阿，可以建一个工程直接操作数据库，而且使用比较方便
这就是图片

首先新建工程，选则java－ desktop application
然后next，再在choose application shell中选择database application
再next
最后在master connection选择new database connection
我使用的是MySQL，所以要选择MySQL，
然后填写一下URL jdbc:mysql://localhost:3306/database_name
和user name，password
最后finish，run就ok了
很爽阿，可是在ubuntu上，我没有使用系统默认的皮肤，于是就产生这样的bug
