NetBeans Weekly News # 191 - May 02, 2012

发表于 2012-06-15

社区


自2011年 NetBeans 使用提升 80%


在 Java IDE 市场，最新的 ZeroTurnaround 开发报告显示 NetBeans 自 2011 年的使用提升了 80％！




为 Jax Awards Top Java Ambassador 投票


NetBeans 梦之队队成员 Bruno Souza 和 Adam Bien 被列为2012 Java 大使！花几分钟，看看通过的候选人，并根据情况投票。




新的 NetBeans 插件：SVG 2 FXML


来自 NetBeans 梦之队的 Toni Epple，创造了一个新的 NetBeans IDE 插件，把 SVG 转换为 JavaFX FXML。尽在他的 YouTube 和报告中的全部代码列表。


 



教程



Mercurial Queues 指南


本教程演示了如何在 IDE 中执行基本的补丁任务，使用 Mercurial 版本控制软件通过标准的工作流程来管理补丁。




入门插入到项目中的“Find in Projects”对话框


在 NetBeans 中通过扩展的对话框可以搜索到扩展项目中新的 API。




YouTube：NetBeans IDE 中的 TestNG


在 NetBeans IDE 中创建和调试 TestNG 测试类。在视频中，创建一个简单的 TestNG 测试，在测试类中创建一个数据提供者来支持测试中使用到的参数。


 



文章



PrimeFaces 3.3 正式发布


Prime Teknoloji 高兴地宣布 PrimeFaces 3.3 版本发布。有 10 个 issue 在 3.3.RC1 被修正。




Javeleon 2.0 发布


Javeleon 团队很高兴地宣布了新的更好的 Javaleon 2.0 类的改进。在开发过程中重新加载工具时无需通过重新启动应用程序，从而提高您

的工作效率。



博客



精确找出 GUI 发生了什么


来自南非的 Hermien Pellissier 介绍了有关 NetBeans GUI Builder 中的方便提示。




整合进度栏到向导中


如何通过向导 APIs 来整合进度条在一个长时间运行的任务中？请进一步阅读。
