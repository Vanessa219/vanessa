NetBeans Weekly Newa-Issue # 426 - Feb 09, 2010

发表于 2010-02-12
















































2010-02-11NetBeans IDE 6.8补丁1发布
NetBeans团队已经发布了新的补丁，这是NetBeans IDE 6.8的更新。该补丁修正了一些错误。安装和运行NetBeans IDE 6.8bug就可以被修复。使用IDE的插件管理器也可以获取补丁。		Edit
2010-02-11PHP新功能中的反馈信息：Error badges
NetBeans PHP开发小组希望在NetBeans 6.9中添加反馈信息这一实验性的功能：Error badges。如果您有任何错误的PHP文件，该文件在项目窗口中可以被直接标记。让开发小组知道它是否是一个有用的功能？		Edit
2010-02-11JavaFX应用支持RTSP
自从JavaFX SDK 1.2版本以来，实时流协议（RTSP）的支持便可被使用。这份文件阐明了哪些平台支持此功能，并在您的应用程序中提供了可用流媒体的基本推荐。		Edit
2010-02-11Java和NetBeans的Swinging和Tweeting
在Packt出版指南中，作者Alfonso Romero概述了如何使用Java和NetBeans，而不是使用标准的网络接口来建立一个Twitter的自定义应用程序。		Edit
2010-02-11在Java应用程序中添加一个文件选择器
了解如何使用javax.swing.JFileChooser组件和 NetBeans GUI Builder在Java应用程序中添加一个文件选择器。		Edit
2010-02-11JavaFX设计器——JavaFX的一大步？
JavaFX设计器对于NetBeans 6.8来说是一个新的可视化编辑器插件（类似于项目Matisse中的Swing）。DZone采访了JavaFX的开发小组成员，了解到了用户对这个工 具的反馈信息。更多像JavaFX设计器的工具可能导致JavaFX应用程序的高增加，例如在2010年冬季奥运会中的奖牌实物应用网站。		Edit
2010-02-11视频：2010年Jfokus ——NetBeans IDE和JavaFX
在Epsilon的开发人员，也就是 瑞典的技术和系统顾问组织都是JavaFX和Netbeans IDE的爱好者。在2010年的jfokus展会上，他们演示了一个项目并且讨论了这两种技术的使用。		Edit
2010-02-11创建您自己的窗口系统
你知道你可以根据NetBeans窗口系统的API创建一个自定义的窗口系统吗？一个可以集 成其他NetBeans的方法？在来自Geertjan Wielenga和Toni Epple的博客中你可以了解到更多信息。		Edit
2010-02-11使用NetBeans IDE连接Oracle数据库
Arun Gupta的第107届“日积月累”中介绍了如何使用NetBeans IDE连接Oracle数据库并优于所有Java EE应用程序开发，Rails和其他IDE所提供的全部特性。		Edit
