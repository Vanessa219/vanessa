API 功能简介

  • 网络：HTTPS 请求、文件上传下载（保存）、WebSocket（仅一个）
  • 媒体：图片、录音（1分钟）、语音播放、后台音乐播放、音频、视频
  • 文件：保存本地、获取本地已保存的文件列表、获取本地文件的文件信息、删除本地存储的文件、新开页面打开文档，支持格式：doc, xls, ppt, pdf, docx, xlsx, pptx
  • 数据缓存：增、删、改、清空
  • 位置：使用腾讯地图
  • 设备：获取系统信息、获取网络状态、监听重力、罗盘数据（5次/秒）、拨打电话、扫码
  • 其他：微信登录、获取用户信息、支付、客服、分享、生成各页面二维码
