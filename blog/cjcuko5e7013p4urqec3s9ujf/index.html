回归 Firefox 4, 喜欢了喜欢的插件

发表于 2011-03-26

在 2011-03-02 下定决心使用 Chrome，并安装了一系列插件准备长期备战，谁知就在 03-22 日 Firefox 4 终于发布了，并且获得了不错的好评。


纠结了2天，终于再次换回 Firefox。并且安装了如下插件，其余的都卸载了， very good。非常的好用。


Addons Manager:


    插件，皮肤，扩展管理。个人比较喜欢到官网上直接去淘。


Awesome screenshot:


    网页截图，可以进行简单的标识。非常的好用，强力推荐。


DownThemAll:


    下载管理。一直都用它，习惯了，速度也不错。


Firebug:


    开发人员都知道，不是开放人员的话，也用不上 ^^


Google Dictionary and Google Translate:


    翻译。配置好后源和目标，在页面中双击单词便可获取目标语言。


Google Toolbar for Firefox:


    Google 工具栏。每天都要使用 Gmail 和 Google bookmark，一个插件搞定。注：在 Addons Manager 上搜死都搜不到，要到官网上按照。


Java Console:


    。。。


Page Speed:  


    分析网页加载状况，并给出相关建议及优化后的文件。直接点击还可跳转到相应优化的文档，很有用。


YSlow:


    为什么慢？ Yahoo 告诉你。按照著名的 Yahoo 7 大类中的 35 个最佳实践 为你的网站进行评估和提示。


SeoQuake:


    SEO。对 SEO 了解甚浅，大概看了下，感觉不错，先收藏着。有空好好优化下 b3log solo 和 b3log symphony.


dynaTrace AJAX Edition Agent:


    监视整个页面，然后做出详细的分析。以下废话，可略过。



  • 开始使用 Leak Monitor, 一开始没看懂。后来一知半解，但老弹出窗口，烦，卸载了。



  • 后来使用 JavaScript Debugger(Venkman)，貌似不兼容，一用插件就报错，无奈之卸载。



  • 最后看了下以前安装过的 dynaTrace (for IE)，下载新版本，重新安装。Firefox 也被强行安装上此插件，但一用还是有问题。不过还好 IE 上正常。花了半个多小时用，看着详细的分析结果，还是半知半解。但感觉很强大，很有用（难道是没用懂的幻觉？）。
