2010-12-14NetBeans IDE 中 Cobertura 对 Maven 项目的支持
NetBeans IDE 目前通过 Cobertura Maven 插件可对 Maven 项目的代码覆盖分析进行支持。了解如何在 IDE 中显示代码覆盖分析的测试结果，可参见NetBeans RCP applications on Maven。		Edit
2010-12-14NetBeans 7.0 对 PHP 编辑器性能方面有提示
NetBeans PHP 团队一直在对 NetBeans 7.0 中的 PHP 编辑器在性能方面的 issues 进行修复。（下载每晚构建的版本可获取其中的变化。）要继改进高编辑，团队希望得到更多在性能方面的使用情况。阅读团队博客可获取详情。		Edit
2010-12-14插件：Gmail 的通知
在 NetBeans IDE 中，当有新的 Gmail 邮件时，该插件可进行通知。		Edit
2010-12-14专访：Caoyuan Deng 谈 NetBeans 中的 Scala 插件
开发人员 Caoyuan Deng 谈论了有关 NetBeans 中 Scala 插件的起源，当前状况及项目远景。		Edit
2010-12-14瑞士空中交通管制起飞基于NetBeans平台
去年，SkyGuide，瑞士航空软件公司想要通过 NetBeans 平台培训课程，把其部分应用移植到 NetBeans 平台上。在这篇后续文章中，软件工程师 Julien Piaser 介绍了Crystal，一个基于 NetBeans 平台的应用程序:空中交通管理信息。		Edit
2010-12-14垃圾邮件止于此！Holger Stenzhorn 的专访
NetBeans 社区的版主 Holger Stenzhorn，叙述了他是如何与其中一个邮件账号（NetBeans 邮件列表中的账号）中的垃圾邮件战斗的日子。		Edit
2010-12-14新版本的 Kojo（Scala 学习环境）可用
在 NetBeans 平台上创建 Scala，Kojo 是一个学习环境，可以教导孩子编程的基本知识，并帮助他们去探索数学，科学和艺术的概念。可从给人留下深刻印象的开源项目的最新发布中了解更多相关的新功能。		Edit
