11个视觉差滚动效果的 HTML5 网站

发表于 2012-06-27

以前的视觉差只是简单的背景和前景滚动速度差异所致，但这些 Demo 呈现的是另一种感觉，非常 Cool。


视觉差滚动效果意为多种背景层可以横向及纵向各自滚动。这个惊人的视效最早出现在 Nike 2011年的 Nike Better World 项目里，自此这个效果广为沿用，时至今日，诞生了很多不错的作品。在此，与各位读者分享11个使用视觉差滚动效果的企业网站模版。绝对可以让你们大饱眼福。


来源：Wabbaly
编译：Antonia Huang@DamnDigital（转载请注明来自DamnDigital）



关于作者：Sergiu Naslau
网页兼平面设计师，钟爱视觉交流。现于北丹麦大学学院（University College Nordjylland）攻读多媒体设计与沟通专业。wabbaly.com 是由他创建的，用以分享交流一系列视觉作品。同时读者可以通过他的 twitter和facebook与之交流互动。



Bit This


Bit This 是一家位于西班牙马德里的代理公司。他们用视觉差滚动效果以展示公司的规制度和行为准则。5个简单的原则就将其品牌和客户包括你们紧密相连。



New Zealand


The New Zealand 是一家介绍新西兰旅游的网站。通过网站，潜在客户可以了解到新西兰的多面性，展现了无比精彩的梦幻之旅。他们用视觉差滚动效果营造出奇妙的旅行经历，仿佛身临其境，只能感叹他们将这个效果和营销策略结合的如此天衣无缝。



Diablo Media


Diablo Media 是一家线上营销代理公司：为广告主和出版商发布产品和作品。他们的视觉差滚动效果很有趣，从一页到另一页的切换，虽然速度不快做的也很简单，不像 the New Zealand 那般“鸿篇巨制”，但是还是足以吸引众人的眼球。



360 Long Road Zurich


之前发布的《Inspiration：40个高清图像网页设计案例》以及这周一发布的《TheFWA：360°Langstrasse Zurich》里都有推荐过。
通过一个网站，就可以让我们疯狂领略到瑞士的每个角落。跟着警察巡逻车，沿途可以看到最老牌的本土生意、普通的家庭生活、企业家们，还有多彩的鸟儿。视效 太震撼了。像这样的作品真的让人感叹 HTML5 太强大了。我个人很喜欢鸣叫声的处理方式，以及他们对本土生意的描述。看完了这一切，脑中立马浮想联翩，激发出了很多灵感。



Netlash bSeen


Netlash BSeen 是由 Netlash 创建人和线上营销代理 bSeen 合并之后新创建的一家代理公司。网站的文字部分描述了公司合并的过程以及客户解答。简单却不失效果。



Ito Sio Ceni


Ito Sio Ceni 是一家波兰的广播公司（至少我是这么认为的）。可以看到视觉差滚动效果应用于麦克风以及页面的切换，视效与网站内容整合的相当贴切。



Sullivan


Sullivan 是一家提供从各个方面提升用户接触度服务的代理公司，通过视觉差滚动效果展现其最新作品。简单却能如实反应出他们的正义感。



Nike for a Better World


Nike Better World 是第一个使用内容滚动串联图版的网站。对 Nike 来说可是个大飞跃，因为之前他每年都习惯使用 flash。点击这里，你也可以通过其他网站看到这一革新效果的使用。



Iutopi


Iutopi 是一家品牌、平面设计和网页设计代理公司，网站做的非常棒。通过视觉差滚动效果展现了一个精彩故事。



Cultural Solutions


Cultural Solutions 是英国的一家企业管理代理公司，视觉差滚动与泡影效果相结合，绝对冲击视觉神经。他们虽然不像其他网站那样做的复杂，但是很美观。



Go Up


Go Up 是一家意大利广告代理公司，运用视觉差滚动效果展现其涉足的各个领域。
