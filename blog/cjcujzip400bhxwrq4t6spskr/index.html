2011-01-25NetBeans 插件：SCSS（“Sassy CSS“）支持
SCSS 是两种对 Sass 可用语法中的一种，它扩展了 CSS，在基本语言上更加强大和优美。在这里可了解到更多的 Sass。这是 NetBeans 社区贡献的插件，添加了对 SCSS 文件编写的支持。
2011-01-25APIGen 插件：PHP 项目 API 的生成器
APIGen NetBeans 插件把 API 文档生成器 APIGen 集成到了 NetBeans IDE 中，它允许你在 PHP 项目中生成 API 文档。

2011-01-25Revived: NetBeans 平台中的罗马尼亚本地化
