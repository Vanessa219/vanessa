『书单』小程序 symphony-weapp 开源了

发表于 2017-01-06

稀饭请 star https://github.com/b3log/symphony-weapp

symphony-weapp

Symphony 社区平台的微信小程序，提供一些实用工具服务，比如『书单』。

登录

  • 对 md5.js 进行封装，使其可以在小程序中进行使用
  • 微信提交登录请求
  • 使用微信数据缓存存储用户标识以实现类似 Cookie 的作用

书单

  • 调用微信扫码接口读取书籍 ISBN
  • 微信提交书籍信息查询

共享

  • 微信提交书籍共享请求

其他微信 API

  • 设置导航条
  • 导航
  • 交互反馈
  • 网络状态

服务端

服务端接口请参考 Symphony 项目 :)
