网页设计师必用的14个Firefox扩展

发表于 2009-05-22

责任编辑：杜涛作者：网络 Roby Liang   2009-05-21   

  Web Developer Toolbar 

    关于 Web Developer 工具栏我想这里就不用多省笔墨介绍了吧。


    Firebug 
    跟 Web Developer Toolbar 一样，Firebug 也强大的让人有些相视恨晚。Web Developer Toolbar 塞给了我们一个工具栏，虽然也可以靠右键菜单在目标网页中进行操作，不过终归不如工具栏方便。这一点对于有些被 XX 工具栏吓坏了的用户也许很难打接受，当然也有些觉得只要加了工具栏就是流氓软件的朋友们。Firebug 让这些 Firefox 看到了希望，不强加工具栏，也可以编辑、调试和监控任何网页上的 CSS、HTML 和 Javascript。


    Console2 
    从Firefox1.5 版本开始，JavaScript 控制台就不支持 CSS 了，只能用于错误调试。这个扩展可以让用户使用不同的标签来标注不同的调试类型。不过说实话，有了 Firebug 之后，就没有装这个扩展的必要了。


    CSSViewer 
    这是笔者最喜欢的扩展之一了。当初将她上传到 mozine update 完全是因为作者漂亮的预览图，可一试用却发现自己还是太肤浅了，使用这个扩展查看网页中的各项 CSS 属性简直太方便了。


    User Agent Switcher / IE Tab / OperaView 
    在这里偷了一下懒，将三个扩展放到了一起。借助这三个扩展可以使 Firefox 很方便的更改浏览器 UA 和调用 IE 或者 opera，测试自己编写的网页在不同的浏览器下不同的表现。


    ColorZilla 
    利用 ColorZilla 大家可以从浏览器中的任一点读取色彩信息并粘贴到其他应用程序，也可以缩放正在查看的页面，测量页面任意两点间的距离，还可以从预定义的色彩中中选择颜色 并保存到自定义调色板。DOM 侦测功能让大家可以快速、容易地获得关于 DOM 元素的各种信息，设计见面的朋友们一定要试一试她。


    ViewSourceChart 
    ViewSourceChart 很体贴的为 HTML TAG 边界相关定义人性化的提供了图形化操作界面，是网页设计初学者的一款利器。


    Tamper Data 
    Tamper Data 在 Mozine 论坛很少被提起，不过老衲认为这款扩展绝对值得高度关注。借助 Tamper Data 我们可以密切的监视，记录甚至是修改 Firefox 发出的每一次 http 和 https 的请求与响应，堪称是我们的安全卫士。对于网页设计者来说，这款扩展可以帮助我们检测自己写的网页中的超链接的健康状况。


    JavaScript Debugger 
    一款在 Mozilla 平台下的脚本调试工具，又被称做 Venkman。但是要做好心理准备的是很多用户反应这个扩展在 Firefox 下并不是特别顺手，我也没有做过详尽的测试，所以在这里也不好多发表意见。不过，看起来很不错。


    FireFTP 
    我想平时使用 Firefox 的网管 GG 们应该很少有人使用 Firefox 做为 FTP 终端来管理自己网站服务器端的目录吧。实话说 FireFTP 的效果很差，尤其对于我们中文用户来说还存在着中文路径的 BUG（其实我也不知道在最新的版本中到底有没有根治这个 bug）。不过从积极的意义来看，FireFTP 的出现毕竟为我们提供了一种在 Firefox 上管理网站服务器目录的可能，让我们试目以等待吧。


    MeasureIt 
    MeasureIt 提供的功能特简单，就是测量当前网页中任意部分的长度与宽度，由于她太低调，所以到今天都还没有引起人们足够的重视，对于有这些的需要可是不想安装 Web Developer Toolbar 这样的大家伙的朋友们，MeasureIt 是个不错的选择。


    Edit Cookies 
    Edit Cookies 似乎在国内也比较少的被人所提及，这款扩展提供了一个图形界面来管理网站的 cookie，甚至延长 cookie 的有效期，各位站长也可以借助这款扩展来查看自己网站的 cookie 情况。
