2010-09-212010年的 JavaOne：NetBeans！
2010年 JavaOne 大会将于本周在旧金山举行。找出您可以加入的 NetBeans 团队并成为其中一员，就可以与当地的 Java 冠军，Java 用户组领导和 NetBeans 社区成员进行畅谈。可通过 NetBeans.org 从本周 NetBeans 相关的会议幻灯片中进行查看。		Edit
2010-09-21NetBeans 旧金山平台培训
现在，您还有时间去注册由 Marakana / Eppleton 在9月27日至29日于旧金山举行的 NetBeans 平台课程。		Edit
2010-09-212010黑莓开发者大会将对 NetBeans 用户提供折扣
黑莓 DevCon 组织者对 NetBeans 用户的特别优惠将延长。从9月27日至30日b，将与其他开发商联系，邀约 RIM 专家，深入探讨更深层次的黑莓平台开发工具。注册特殊代码 DD6RAC 后可享受 849 美元的优惠价。 （现场费用为1399美元）		Edit
2010-09-21印度那格浦尔的新 NetBeans 用户组
在那格浦尔的 NetBeans 用户组是一个全新的，由志同道合的开发者们组成的团队，他们对 NetBeans IDE 和 NetBeans 平台充满了热情。加入该团队，其成员将于9月23日举行首演会议来确定他们的总体目标和行动计划。		Edit
