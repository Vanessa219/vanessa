背景

本本买来快一年了，每天都是上班 Windows，下班 MBP。这次下定决心要好好体验一番 OS X 了。

体验顺序

Alfred

App Store 直接下载即可

以前用过一次，没用明白，然后就给卸载了。这次又静下心来研究一番，终于发现了它的好。
可以先阅读基本使用及配置，然后再进阶 workflow，好有:heart:呀

丢掉鼠标－Mac神软Alfred使用手册
杀手级功能WorkFlows介绍(1)-Alfred使用手册2

首先就是关闭 Spotlight 的快捷键，设置 Alfred 的快捷键

系统偏好设置 -> 键盘 -> 快捷键 -> Spotlight -> 取消［显示 Spotlight 搜索］ 的选中
Alfred -> Preferences -> General -> 选中 ［Startup］ -> Alfred hotkey 设置为 Command-Space

使用的途中，差点就又放弃了。因为不像上文链接中说的那样，无论你输入什么，就只会给你提示 Search Google for 'XXX'
还好及时 Google 了一下，原来是 Spotlight 的索引服务停了。所以只需要运行一下以下命令，本来就已经停了，准确的说运行最后一句就可以了

sudo launchctl unload -w /System/Library/LaunchDaemons/com.apple.metadata.mds.plist
sudo launchctl load -w /System/Library/LaunchDaemons/com.apple.metadata.mds.plis

好开心，又可以愉快的玩耍了～

iTerm

官网 直接下载安装
看上去完全不好用的节奏，算了，都被评为神器了，还是静下心来用用吧。

  1. 浏览一下个菜单和 Preferences
  2. 配置颜色
    2.1 下载 Solarized
    2.2 Preferences -> Profiles -> Colors -> 右下角 Color Presets -> Import -> 选择刚才下载的 solarized/iterm2-colors-solarized/Solarized Dark.itermcolors -> 导入成功后选择再选择 Color Presets -> Import -> Solarized Dark
  3. 配置 vim 配色

贴入以下内容

syntax enable
set background=dark
colorscheme solarized

  1. 配置 ls 配色

贴入以下内容

export CLICOLOR=1

隐藏文件

觉得显示出来好丑，但是 .env .gitignore 都看不到，还是显示一下吧

$ defaults write com.apple.Finder AppleShowAllFiles YES
$ killall Finder
