NetBeans IDE 7.3 补丁1


NetBeans 团队为 NetBeans IDE 7.3 发布了补丁，用于提高稳定性和性能的众多修复。



为了获取修复，先要安装 NetBeans IDE 7.3 并运行它。更新通知将会出现在 IDE 中。单击通知即可安装更新。此外，您还可以通过 NetBeans IDE 插件管理器下载补丁。
