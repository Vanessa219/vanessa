NetBeans插件Portal Pack 3.0 发布

发表于 2009-02-23

【CSDN 2月20日消息】最新Portal Pack 3.0下载地址为：http://portalpack.netbeans.org/pp30/download.html。



Portal Pack 3.0的新特性如下 :

•    支持JSR 286规范 

•    对JSR 286事件设有故事板功能

•    利用Visual Portlet Builder插件可在WYSIWYG编辑器中建立JSF portlet

•    带Spring MVC框架的Portlets

•    使用不同语言（如Ruby/PHP/Groovy）的Portlets，可在Sun GlassFish Web  Space Server and Liferay Portal Server上部署

•    支持Liferay Service Builder

•    目录部署: 由Portal Pack支持，改变JSP/HTML/JS文件时不需要重新部署portlet程序

Portal Pack 3.0支持在以下服务器上的部署：

•    Sun GlassFish Web Space Server 10.0 (也称作Project WebSynergy)

•    Liferay Portal Server 5.1.x/5.2.x

•    Open Portal Portlet Container 2.x



预知详情，请查看http://blogs.sun.com/satya/entry/portal_pack_3_0_makes
