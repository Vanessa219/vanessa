偶决定开始使用 Chrome 作为默认浏览器了

发表于 2011-03-02

Chrome 从发布到至今，虽见证了它的辉煌和优势，偶尔也小用一下，可是一直舍不得放下习以为常的 Firefox。


直至不久前看到《15个对Web设计和开发有用的Chrome插件》，以及越来越不能忍受 Firefox 的启动速度。



于是就在昨天安装了选择性的安装了一系列开放和日常所需的 Chrome 插件：


1. Chrome SEO：显示当前网站与搜索引擎优化相关的信息，包括反向链接，Alexa排名、Google PageRank排名等流量值数，在社会化书签网站中的热门程度，对 key 的建议等等。


2. Webpage Screenshot：网页截图。可截取整个页面或可见区域，截取后可进行简单的编辑


3. Firebug Lite：见图就只真相


 


4. MonitorTab: Speed Tracer。


使用前须在



的目标后添加 –enable-extension-timeline-api


为了立即体验它，特在本博客和 b3log 社区上进行了使用，good，貌似没有任何的性能问题。


 


5. Pendule:包含了很多对于开发者来说很有帮助的任务选项。比如重新载入或是禁用 CSS，查看 JavaScript 代码，隐藏图片，取色器，测量尺，查看页面源代码，以及多种编码验证工具等。


开始还安装了个专门的 Eye dropper: pick color 和 web developer，当发现它后就把他俩都卸载了。因为 Pendule 确实真的很方便。


 


6. Google Bookmarks: 同步 google 书签，推荐一下地址的，其它的都不太好用。


https://chrome.google.com/extensions/detail/fhligfgeenbkgbiehjkfpkljkaonhfah?hl=zh-cn


 


7.Google Mail Checker：gmail 邮件提醒


 


随便试用后，决定开始试着习惯 Chrome，也相信它可以越做越好。


不过某天 Firefox 4.0 正式发布，如果真的很不错，我还是可以考虑换回 Firefox ^^
