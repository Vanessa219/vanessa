NetBeans Weekly News Issue # 290 - July 15, 2014

发表于 2014-07-03

NetBeans IDE 8.0 满意度调查
NetBeans 团队想知道新的 NetBeans8.0 为你工作的如何！是不是一切如你所期望的，还是有需要改进的地方？花费5分钟，在这个快速调查中给 NetBeans 团队你的反馈！


插件：Praxis LIVE 渲染器
通过视觉体验的几下点击， NetBeans IDE 的外观被 PRAXIS Live 重新创建，在 NetBeans 平台上创建了一个视频投射应用程序。


帮助 Java 性能优化的9个工具
一篇新的文章列出了许多分析工具，包括了 NetBeans Profiler，以帮助你提升你 Java 应用程序的性能。


NetBeans 平台上的捕捉及接入系统
在巴西，开发商宣布了一个在 NetBeans 平台上创建的新框架，它支持多样化媒体捕获和结构化文档的生成，专注于视频，音频，幻灯片和屏幕捕获。


NetBeans 平台上的复合材料计算器
在德国德累斯顿大学的开发者们宣布了一个在 NetBeans 平台上创建的新的免费的应用程序，它用于航天工程的，主要专注于复合材料的计算。


NetBeans 梦之队：NetBeans IDE 中我们最爱的功能！
Maven，性能分析，以及更多… Zoran Sevarac, Kirk Pepperdine 和其他 NetBeans 梦之队成员与您分享他们对 NetBeans 最喜欢的功能。


NetBeans 课堂：如何教 JavaBean？
在加拿大教授 Java 的 Ken Fogel，讨论了在课堂上的讲解 JavaBeans 时可以采取的办法。


视频：根据代码覆盖来标识出冗余代码
Adam Bien 为你展示了如何在 NetBeans IDE 中使用工具来帮助检查代码覆盖，同时能标识出冗余代码。


B3log Wordman 1.0.0 发布！
沃德曼（Wordman）是一个移动端的背单词应用，使用 Cordova 开发，目前发布了 1.0.0 的 Android 版本，欢迎大家下载试用。
