从Web0.2到2.0 那些与时俱进的Web技术

发表于 2009-08-02



 

 感谢ydj9931
的提供 |




我们初识互联网之时，它正处于WEB 0.2的时代。计数器，GeoCities和留言板统治着WEB世界。当时CSS只是一门新技术。订阅此页是JavaScript最大的应用。而最终，IE 4美化了桌面浏览器的界面。


时光流转，十多年之后，似乎一切都不同了。Ajax遍布网页，JavaScript框架使得网页动画轻而易举，手持设备给我们随时随地的提供了互联
网的访问方式。不要被表面现象迷惑了，我们认为某些技术十年前就被抛弃了，而实际上它们仍然存在于网上，只不过是以另一种姿态示人。




WEB 0.2：计数器；WEB2.0：FeedBurner订阅者标
志


计数器是衡量一个网站受欢迎程度的最终手段，因为黑白相间的曲线不会骗人。如今是2009年，我们把计数器从网站点击率迁移到了企业联合组织化点击率。别骗自己了：计数器依然存在而且状态不错。


WEB 0.2：留言板；WEB 2.0博客评论


原来，90%的情况下，我们都是用留言板来给站长拍砖。现在，丑陋的CGI留言板已经被取代，我们选择了在博主博文的评论区来拍砖。


WEB 0.2：GeoCities；WEB 2.0：MySpace


让我去抨击GeoCities是很难的，因为我在那儿建立了我的第一个网站，虽然它只是一个业余作品。早在GeoCities实现所见即所得的编辑
器之前，站长们就在使用劣质的HTML标签，可怕的GIF和强制大小的图片来实施其网页恐怖分子的行径了。如果我没有提到GeoCities这个名字，你
肯定会认为我是在讲MySpace，对吧？


WEB 0.2：Frames；WEB 2.0：DiggBar


Frames一直困扰着从搜索引擎而来的访问者，他们搞不明白当前页面是否是复合框架的一部分。而把框架中的页面加入书签又会如何呢？你总是会得到
同一个index页。这太恐怖了。DiggBar把Frames作为一个实现简单投票和评论的工具从新带回了互联网。实际上他们带回来的是WEB的瘟疫。


WEB 0.2：邮件列表；WEB 2.0：RSS订阅/谷歌快
讯


邮件列表让你可以无需花费多少力气就获得每日更新。它的唯一问题就是你的邮箱中最终会留下成百封的未读邮件而你再也不会去读它们了。不知道你如何，但是我使用RSS的时候总是会陷入这种境地。还好RSS不会撑爆你的邮箱。
