NetBeans Weekly News # 230 - Feb 26, 2013

发表于 2013-02-27





















































 NetBeans IDE 7.3 发布
NetBeans IDE 7.3 诞生了！7.3 的发布使开发人员能够创建和调试基于最新的 HTML5，JavaScript 和 CSS3 标准的 Web 移动应用。开发人员可以使用及时页面预览器查看丰富的 Web 移动应用开发中页面的预期效果，Chrome 浏览器集成，CSS3 样式编辑器，基于最新的 Nashorn JavaScript 引擎重写的 JavaScript 编辑器，一个新的 JavaScript 调试器等等。		 
 NetBeans IDE 页面功能重写
这篇文章概要了 NetBeans IDE 在数星期前被重写的主要特点！Java SE，Java EE，HTML5 和更多：阅读所有的工具和实用程序，使用 NetBeans IDE 可加速您的开发。		 
 SDTimes：NetBeans IDE 7.3 对 JavaScript 扩展的支持
尽管 NetBeans 一直定义为 Java IDE，但对其它语言的支持总在不断探索和进步。NetBeans IDE 7.3 已经发布，它带来了一系列新的对 CSS3，HTML5 和 JavaScript 的支持。		 
 InfoWorld：甲骨文更新 NetBeans IDE 中的 HTML5
NetBeans IDE 7.3 “允许开发人员使用相同的 IDE 来组装 HTML5，使用构建后端服务来连接他们的 Web 和移动应用”Bill Pataky，甲骨文公司产品管理工具和框架的副总裁表示。		 
 jaxenter：NetBeans IDE 7.3 拥抱 Web 的到来，支持 HTML5
“Web 开发功能引入到流行的 IDE 中展现了我们当前生活的融合发展。不再可能避开对 HTML5 和 JavaScript 应用开发的喜欢，Oracle 决定拥抱他们。”		 
 Anahata 宣布 NetBeans 7.3 是首选的 IDE
NetBeans 是一个伟大的 IDE，用来开发最新标准的 Java SE 和 Java EE 的 Java 应用程序。它有一个充满活力的开源社区，可快速响应我们发现的功能需求。		 
 自定义和自动化的搜索/替换
Custom hints (在其他 IDE 中又名 custom inspections 或者 quick fixes）在 NetBeans IDE 7.1 中已实现。他能匹配 Java 代码并提供转换。您可以基于 AST 创建简单的重构，无需编码。一个典型的示例是类型安全的 API 迁移，而不是容易出错的基于纯文本的“查找和替换”的方法。		 
 2013 JavaOne 印度大会：CFP Open
JavaOne 印度大会于5月8号至9号在 Hyderabad 国际会展中心举行。与往常一样，这两天将装得满满的，更多精彩内容值得您期待，从技术会议的主题演讲到动手实验和演示。
