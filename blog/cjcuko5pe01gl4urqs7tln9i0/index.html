计算机常用10句日语

发表于 2008-03-16

汉字：日本　語　を　入　　　力　　　する　例
    にほんご　を　にゅう　りょく　する　れい
解释：输入日本语的例子

汉字：日本　語　計　算　機　の　学　習
     にほんご　けいさんき　の　がくしゅう
解释：日语计算机的学习

汉字：アプリケーション　の　基本　　操　　作
　　　          　　　　　　の　きほん　そう　さ
解释：应用程序的基本操作

汉字：こ　の　章　　　の　内　　容　　　　　　
      こ　の　しょう　の　ない　よう
解释：这章的内容

汉字：ワードパッド　を　起　動　　する　には
                       　　  を　き　どう　する　には
解释：为了打开word

汉字：保　存　　した　文書　を　開　く　には
      ほ　ぞん　した　しょ　　　ひらく　には　
解释：为了打开保存的文档

汉字：ウィンドウ　の　サイズに　合わせて
　　　         　　　の　　　　　　あわせて
解释：符合窗口的尺寸

汉字：Windows　で　利　用　　する　アプリケーションは
          　　　　　で　りよう　　する
解释：使用windoes应用程序

汉字：スタート　メニュー　内　の　移　動
                   　　　　    ない　い　どう
解释：在开始菜单内移动
