项目新闻



NetBeans 6.7里程碑2现在可以下载了！



NetBeans 团队兴奋的发布了NetBeans IDE 6.7的里程碑2。发布中包括了几个改进和增强的特征：Mave 和 C++ 的支持，分析工具，SVN组件等等。
伴随着发布，IDE目前能给用户选择激活他们想使用的功能。增强自我诊断的工具可以让用户审查问题的表现情况，或者发送一份报告给 NetBeans 团队进行分析。MAC OS X的外观和风格同样也能得到很好的更新。了解更多关于发布中新的和值得注意的信息，并给我们您的 反馈！



文章



PHP IDE的测试：为什么使用 & 如何选择



Smashing 杂志的 Alexander Makarov 在这篇文章中对几个 PHP IDE 进行对比，其中 NetBeans IDE在编辑工具，代码完成，响应能力和此外的其它功能中获得了较高分数。




高水平 Java 开发者的小测验



最近 SUN 开发者网络项目（SDN）Java 问答调查了过去采访过的领先Java开发人员收集的问题——可能是挑战，告知，招待，娱乐和刺激。在 Java 社区中您能够找到人们好奇和精神乐趣所在的问题！



博客



NetBeans 6.7——Ergonomics IDE 的测试



赞成 NetBeans 的作者 Adam Myatt 在 NetBeans 6.7里程碑2的发布中以一个全新的功能作为实验——当 IDE 加载时你可以通过选项选择您只需要的功能。




HeapWalker 对 OQL 的支持



NetBeans 6.7里程碑2 加强了 HeapWalker 中对 OQL 的支持。NetBeans Profiler 团队博客中有一个简单的增强 OQL 整合的演示链接。



社区



NetBeans 管理委员会的最后投票



今天是 NetBeans 管理委员会选举投票的最后一天。这里再次向您列出优秀的被提名者名单：
Toni Epple Ryan de Laplante
Fabrizio Giudici Tonny Kohar
Kristian Rink Tom Wheeler
从他们的个人资料中进行详细了解后，您可以从中选出最多3名候选人。选举截止日期为最后一个时区的三月二日星期一午夜。选举结果和新的委员会将在3月9日进行宣布

在 Java Swing 应用程序中使用 Hibernate



使用 NetBeans IDE创建和部署一个显示数据库中数据的 Java Swing 应用程序。这个程序使用 Hibernate 框架作为持久层来检索关系型数据库的 POJOs（老式的Java对象）。

在 NetBeans IDE 中创建手机游戏
