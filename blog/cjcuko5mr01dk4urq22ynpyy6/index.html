申请 js.org 二级域名

发表于 2017-01-28

继 https://hacpai.com/article/1485173842388 后，我本应该是去升级我的 webpack，但是但是，我的注意力被 js.org 这个域名吸引了。本宝宝也想要～～～～ I Want I Want I Want it…

原来还是蛮简单的，参照 https://dns.js.org/ 一步一步往下做就可以了。

  1. 登录 GitHub，参照 GitHub Pages 进行配置
  2. 检测你需要的 foo.js.org 是否存在？ So lucky，我要的还在 :star2:，在想要不要帮 hacpai，b3log，sym 也注册一个。算了，他们又不是 js…
  3. 在你设定的 GitHub Page branch 上新建 CNAME 文件
  4. 到 GitHub repository 上 PR 一发
  5. 坐等 merge

但是，结果还是来来回回使用蹩脚英文沟通了几次 https://github.com/js-org/dns.js.org/pull/1025 。不过结局蛮好的，除夕夜前 https://vanessa.js.org 已经可以访问了哦。本来今天下午要修剪一下感觉长了10年的花草 - https://vanessa.js.org 。哎，最后还是做了一个把自己丑哭的新年首页，不管了，我要洗白白过大年去了，还要准备好晚上的红包～～～

祝 HPer 鸡年大吉
