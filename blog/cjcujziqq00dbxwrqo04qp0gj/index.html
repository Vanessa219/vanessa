2010-10-18


下载 NetBeans IDE 7.0 里程碑 2
NetBeans IDE 7.0 里程碑 2 现在已可供下载。这次建立的稳定开发重点包括新功能及对 JavaEE，GlassFish，WebLogic 等的改进。		Edit
2010-10-18特性插件：PHP Nette 框架
在 NetBeans 目前特性插件的项目中提供了对 PHP Nette 框架支持，语法高亮，模板以及更多！		Edit
2010-10-18简化 Web 服务生成
REST 简化了 Web 服务的创建。NetBeans IDE 通过提供的工具对 REST 进行了近一步的简化。阅读 Jeff Rubinoff 即将发表的更深层次的报告。		Edit
2010-10-18国际化 GUI 表单
本教程将向您介绍如何使一个单独的表单乃至整个项目进行国际化。		Edit
2010-10-18NetBeans IDE 中 PHP 的代码模板
了解有关 NetBeans IDE 中易于使用的 PHP 代码模板。如何使用它们和一些常见的用例。		Edit
2010-10-18在伦敦的 NetBeans RCP 培训
上周，Jaroslav Tulach, Toni Epple,和 Geertjan Wielenga 在伦敦 SkillsMatter 介绍了 NetBeans 平台的免费活动。		Edit
2010-10-18在Bielefeld 的 NetBeans RCP 培训
在德国比勒费尔德大学的生物信息学学生正在利用 NetBeans 平台研究他们的工作，在这里可读到所有相关内容。		Edit
2010-10-18水族馆博客对 NetBeans 7.0 里程碑 2 的报道

2010-10-18Vaadin 插件的改进
Vaadin 插件对 NetBeans IDE 来说是一个很大的发展，基于来自 Vaadin 团队的请求。在这里可读到所有相关内容！		Edit
2010-10-18NetBeans 平台和 Cookies
Cookiemonsters 注意了！来自瑞典的尼克拉斯低频向您介绍了“Cookies”在 NetBeans 平台上的概念。		Edit
