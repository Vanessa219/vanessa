OpenOffice.org 更新至 3.0.0

发表于 2008-10-12

才把公司里的openoffice给卸载了，本来想装2007的，可是怕慢死，于是装了个word2003.。。。


星期一去体验一下openoffice3.0.0


 


 


 


流行的开源集成办公套件 OpenOffice.org 于今日更新至 3.0.0。OpenOffice.org 包含文字处理、电子表格、演示制作、图形编辑及数据库等多个组件，可满足各方面的需要。OpenOffice.org 3.0.0 版在核心方面包括以下 14 项重大更新：



  1. Mac OS X 支持

  2. ODF 1.2 支持

  3. Microsoft Office 2007 Import Filters

  4. Solver

  5. 图表得到了增强

  6. 改进了 Draw 和 Impress 中的裁剪功能

  7. 电子表格支持多用户协同工作

  8. Calc 支持 1024 列

  9. Writer 支持同时显示多个页面

  10. 改善了 Writer 中的注解功能

  11. 全新的图标集

  12. 新的启动中心

  13. Impress 支持原生表格

  14. 增强了 XML 支持并更新了 XSLT based filters


OpenOffice.org 3.0.0 还包括许多细微改进，如 PDF/A 支持、新的图表组件、VBA 支持等。另外，也有一些新的扩展可用，如 PDF Import Extension、Presenter Console Extension 等。
